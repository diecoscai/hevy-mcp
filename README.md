# hevy-mcp

Model Context Protocol server for the [Hevy](https://www.hevyapp.com/) fitness API. Exposes the public `api.hevyapp.com/v1` surface as MCP tools so LLM-driven clients can read and (opt-in) write workouts, routines, exercise templates, and body measurements. Full documentation is published with the Phase B release; this placeholder will be replaced by the complete README before the npm publish.
